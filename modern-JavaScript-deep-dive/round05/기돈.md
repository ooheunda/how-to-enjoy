## 14장 전역 변수의 문제점

-   지역 변수의 생명 주기 = 함수의 생명 주기
-   호이스팅은 스코프 단위로 동작한다.
-   전역 변수의 문제점
    -   암묵적 결합 : var를 사용하면 변수 이름이 중복
    -   긴 생명 주기 :
    -   스코프 체인 상에서 종점에 존재 : 전역 변수가 마지막에 검색, 속도가 느리다
-   함수 레벨 스코프

## 15장 let const 키워드와 블록레벨 스코프

-   let
    -   변수 중복 선언 금지
    -   블록 레벨 스코프
    -   var로 선언한 경우 전역 객체의 프로퍼티지만, let의 경우 전역 렉시컬환경의 선언적 환경 레코드에 존재하게 된다.
-   const
    -   블록 레벨 스코프
    -   호이스팅이 발생하지 않는 것처럼 동작
    -   재할당이 금지
